# Шаблоны — наполни их под свой бренд

Бот универсальный: вся «личность» проекта вынесена в текст, а не зашита в код.
Эти четыре шаблона — то, что отличает твоего бота от моего. Заполни их под свой
продукт, и бот заговорит твоим голосом.

| Шаблон | Зачем | Куда переносить в коде |
|---|---|---|
| [`01-brand-voice.md`](01-brand-voice.md) | Кто пишет, каким голосом, что нельзя | блок `{{...}}` в `lib/brand.ts` |
| [`02-content-strategy.md`](02-content-strategy.md) | Типы постов, цели, соотношение | значения в `lib/content-strategy.ts` |
| [`03-castdev-icp.md`](03-castdev-icp.md) | Кто аудитория, что у них узнаём | модули в `lib/castdev.ts` |
| [`04-post-examples.md`](04-post-examples.md) | Эталонные посты-ориентиры | примеры в `queue/` |

## Порядок работы

1. Заполни `lib/brand-config.ts` (или `.env`) — название, сайт, хэндл.
2. Пройди шаблоны 01→04 по порядку: каждый следующий опирается на предыдущий.
3. Перенеси заполненный текст в соответствующие файлы `lib/` (замени `{{...}}`).
4. Положи 3-5 своих лучших постов в `queue/` как ориентир для генератора.

Не нужно «идеально» — заполни как умеешь, бот начнёт работать, дальше поправишь.
