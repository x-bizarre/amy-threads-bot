# Очередь постов

Каждый пост — отдельный `.md` файл с frontmatter и секциями. Бот забирает
файлы со `status: draft`, публикует ближайший по расписанию (см. `vercel.json`)
и меняет статус на `posted`.

## Имя файла
`YYYY-MM-DD-короткий-слаг.md` — дата помогает сортировать, слаг для читаемости.

## Структура файла

```markdown
---
status: draft            # draft = ждёт публикации, posted = уже опубликован
platform: threads
goal: subscribe          # subscribe | discovery | brand (см. lib/content-strategy.ts)
notes: "зачем этот пост, что ловим"
media: []                # опционально: пути к картинкам
---

# Заголовок (для тебя, в пост не идёт)

## Пост 1
Текст первого поста. Если это тред — добавь «## Пост 2», «## Пост 3» и т.д.

## Пост 2
Второй пост треда (опционально).

## Reply-bait
Первый авторский комментарий — публикуется к хвосту треда, поднимает охват
и приглашает к диалогу (опционально).
```

В этой папке лежат два примера-рыбы — посмотри на них и удали, когда наполнишь
очередь своими постами. Голос и темы постов задаются шаблонами в `../templates/`.
